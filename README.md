# react-native-turbo-image

Performant React Native image component

## Installation

```sh
npm install react-native-turbo-image
```

## Usage

```js
import { TurboImageView } from "react-native-turbo-image";

// ...

<TurboImage
  source={{
    uri: 'https://placedog.net/300/200?id=1',
  }}
  width={300}
  height={200}
/>
```
